# Dataset

The dataset used for this project is the TidyTuesday Anime dataset from April 23, 2019.

The full dataset is too large to upload directly through the GitHub web interface, so the project notebook loads the dataset directly from the original source.

Dataset link:
https://raw.githubusercontent.com/rfordatascience/tidytuesday/main/data/2019/2019-04-23/tidy_anime.csv

The dataset includes anime titles, synopsis text, anime type, genre, studio, ratings, popularity, members, and other anime related fields.

In this project, the `synopsis` column was used as the main text field, and the `type` column was used as the prediction label.
